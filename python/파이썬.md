### 연산자
* 사칙연산
   | 연산자 | 내용 |
   | :---: | :--: |
   | + | 더하기 |
   | - | 나누기 |
   | * | 곱셈 |
   | / | 나누기 |
   | % | 나머지 |
   | // | 몫 |
   | ** | 거듭제곱  |

* 복합연산자
  
  위 사칙연산에서 = 붙이면됨

  연산과 할당이 동시에

  ex) a += b --> a = a + b

* 비교
   | 연산자 | 내용 |
   | :---: | :--: |
   | < | 미만 |
   | <= | 이하 |
   | > | 초과 |
   | >= | 이상 |
   | == | 같음 |
   | != | 같지않음 |

* 논리연산자
   | 연산자 | 내용 |
   | :---: | :--: |
   | A and B | A와 B 하나라도 F면 F |
   | A or B | A와 B 하나라도 T 면 T |
   | Not | T -> F, F ->T |

### 컨테이너
  mutable 변경가능 / immutable 변경불가
  * 시퀀스
    - 문자열 : 문자 나열
    - 리스트 : 변경 가능값 나열
    - 튜플 : 변경 불가능 값 나열
    - 레인지 : 숫자 나열

    | 연산자 | 내용 |
    | :---: | :--: |
     | 인덱싱 s[i] | s의 i번째 항목 시작 0 |
     | 슬라이싱 s[i:j] | s의 i 부터 j까지 슬라이스|
     | s[i:j:k] | s의 i 부터 j까지 슬라이스 k 간격 |
     | 결합 s+t | 이어붙이기 |
    | 반복 s*n | s를 n번 반복하기 |
    | 포함 x (not) in s | s의 항목중에 x가 있으면 true, (not)그반대 |
    | len(s) | s의 길이 |
    | min(s) | s의 최소값 |
    | max(s) | s의 최대값 |


  * 컬렉션
    - 집합(set) : 유일한 값들의 모음
    - 딕셔너리 : 키-값들의 모음
